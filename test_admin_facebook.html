📊 Database Schema

The following tables have been created/updated:

  • users: Added role, status, banned_until, ban_reason columns
  • user_reports: New table for post/user reports
  • moderation_actions: New table for admin action logging
  • warnings: New table for user warnings
  • deleted_posts: New table for deleted post archive

🔧 API Endpoints Added

  • GET /api/posts/with-role - Posts with current user role info
  • POST /api/posts/{id}/report - Report a post
  • GET /api/admin/check - Check admin privileges
  • GET /api/admin/users - Get all users (admin only)
  • GET /api/admin/posts - Get all posts (admin only)
  • DELETE /api/admin/posts/{id} - Admin delete post
  • POST /api/admin/users/{id}/ban - Ban user
  • POST /api/admin/users/{id}/unban - Unban user
  • GET /api/admin/stats - Platform statistics
