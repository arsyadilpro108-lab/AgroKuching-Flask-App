Understanding how files are organized

All files have been organized into proper folders for better maintainability.

Project Structure

AgroKuching/ │ ├── app.py # Main entry point - START HERE │ ├── Python code/ # Backend code │ └── main_app.py # Flask app with all routes │ ├── JS code/ # Frontend JavaScript │ ├── messages.js # Real-time chat │ ├── home-page.js # Home feed │ ├── profile.js # User profiles │ └── create-post.js # Create posts │ ├── HTML code/ # Web pages │ ├── main-page.html # Landing page │ ├── log-in.html # Login │ ├── sign-up.html # Registration │ ├── home-page.html # User feed │ ├── messages.html # Chat interface │ ├── profile.html # User profile │ └── create-post.html # Create post │ ├── CSS code/ # Stylesheets │ ├── messages.css # Chat styles │ ├── home-page.css # Home styles │ └── ... │ ├── pictures/ # Images │ ├── Default PFP.png # Default avatar │ └── ... │ ├── sounds/ # Audio files │ └── notification.mp3 # Chat notification │ └── agrokuching.db # SQLite database

How to Reference Files

In HTML files:
<link rel="stylesheet" href="/CSS code/messages.css">
<script src="/JS code/messages.js">
<img src="/pictures/Default PFP.png">
In JavaScript files:
const audio = new Audio('/sounds/notification.mp3');
window.location.href = '/HTML code/profile.html';
In Python (main_app.py):
send_from_directory(os.path.join(ROOT_DIR, 'JS code'), filename)

URL Routes

URL File Served Purpose
/ HTML code/main-page.html Landing page
/HTML code/messages.html HTML code/messages.html Chat interface
/JS code/messages.js JS code/messages.js Chat functionality
/CSS code/messages.css CSS code/messages.css Chat styles
/pictures/Default PFP.png pictures/Default PFP.png Default avatar
/sounds/notification.mp3 sounds/notification.mp3 Notification sound
⚠️ Important:
Always use absolute paths starting with / in your HTML files.
Example: /JS code/messages.js not messages.js

Starting the Application

Command:
python app.py
Or use the batch file:
RESTART_SERVER.bat
Access at:
http://localhost:5000
